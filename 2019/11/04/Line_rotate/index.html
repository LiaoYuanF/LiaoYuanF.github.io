前言

题目概述：

Consider there are n points in a plane, let’s denote the set of points as V, any three points in V are NOT collinear. Assuming you have an infinitely long arrow lying on a point P € V in this plane and pointing to the east Now the arrow starts to rotate anti-closewisely with center to be point P. Once this arrow hits a point, say the poin Q after the starting point P, then the rotation origin becomes Q, and continue the process until the arrow is pointing to the east again. If, in the beginning, two points are on the arrow (points to the east), use the right most point to initiate the rotation process.

Given an input of n points using Cartesian coordinate system:

  1. Write a function to return boolean of whether the n points contains any collinear points;

  2. Write a function that accepts an input of a point, say point P, returns boolean of whether starting in this point P, the arrow traverse all the point in V;

  3. Write a function that accepts an input of a point, say point P. this function returns a dictionary of every point and values to be the number of such point to be the center after 2π the arrow rotation;

  4. Write a function to return a dictionary of every point in V, and values to be the boolean of whether the rotation process can traverse all the point in V;

解法

第一小题

第一小题判断V集中是否有三点共线，颇为简单，我直接遍历了点集，直接求出了最多共线点ans，通过判断ans是否大于2进行判断，但要注意不要遗漏垂直的情况

第二小题

这一题涉及到了这条直线的旋转，是这整道题目的核心了，直线不停地旋转并不停地更换旋转中心，我通过点的旋转，之后重新通过俩点坐标来确定直线方程，以达到实现直线旋转的目的。要注意的地方是注意总的2π角度的控制，以及旋转过程中如果出现垂直的特殊情况的考虑

第三小题

这一小题完全是在第二小题的基础上展开的，在第二小题不断旋转的基础上，用字典进行记录作为旋转中心的次数

第四小题

在第二小题的基础上，对所有的点进行遍历，作为旋转中心，如果以此为中心得到的点集与总的点集相同的话，布尔值就为1，输出字典
